# Contact Management System (Python)

## Project Description
This project is a Contact Management System developed using Python.  
It allows users to store, update, search, and delete contact details using a simple graphical interface.

## Features
- Add new contacts
- Update existing contacts
- Delete contacts
- Search contacts
- User-friendly GUI

## Technologies Used
- Python
- Tkinter (GUI)
- SQLite (Database)

## How to Run the Project
1. Install Python
2. Open the project folder
3. Run:
   python index.py

## Project Structure
- index.py → Main application file  
- pythonout.db → Database file
